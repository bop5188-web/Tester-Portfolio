Problem

I needed a working GenAI demo I fully understood — not a group slide deck — that could answer gallery questions through a real chat UI wired to a model.

What I built

  • Coded the full app myself (Streamlit UI, LangChain chain, prompts, OpenAI calls).
  • Ran it locally with an API key for demos and iteration.
  • Debugged UI issues until the chat flow behaved the way I wanted.

How I used AI tools

The product uses OpenAI. For building, I also used Claude, Gemini, ChatGPT, and Cursor to unblock debugging — then I verified the code myself. This page is where I talk about AI depth; my client projects aren’t “AI did it.”

Result

A solo, runnable gallery Q&A app I can explain line by line — proof I can wire a model into a simple product surface and fix what breaks.
